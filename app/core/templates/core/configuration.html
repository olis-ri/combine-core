Combine OAI Server

Combine comes with a built-in OAI server to serve published RecordGroups. Currently, these settings are configured in localsettings.py.

Setting Description Value
OAI_RESPONSE_SIZE How many records to return per OAI response {{ OAI_RESPONSE_SIZE }}
COMBINE_OAI_IDENTIFIER It is common for OAI servers (producers) to prefix Record identifiers on the way out with an identifier unique to the server. This setting can also be configured to mirror the identifier used in other/previous OAI servers to mimic downstream identifiers. {{ COMBINE_OAI_IDENTIFIER }}
