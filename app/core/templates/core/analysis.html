This page provides the ability to view, and run, analysis of Jobs. These special "Analysis" jobs are designed explicitly for analyzing pre-existing jobs, and are not included in the workflow or progression of records for publishing.

New Job

Analysis Analysis jobs are a special kind of job. They do not run in the context of a single Record Group, but instead run in their own unique space in Combine. They are designed for analysis of previously run jobs, allowing for analysis across jobs by merging, running new Validation Scenarios, and/or running different indexing mappers to ElasticSearch. They are intended to be run, used, and discarded where convenient.
